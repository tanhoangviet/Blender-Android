# 0.0.6 VN Support

- Vietnamese app shell strings and startup permission dialogs.
- Blender startup now requests Vietnamese UI locale when the bundled locale is available.
- External USB/OTG hardware keyboard detection hides the soft keyboard and virtual shortcut pad.
- Release build no longer starts Blender with debug `-d`.
- Reduced hot-path logging for drawing, text input, and native input events.
- Faster asset stream copy buffer and explicit install of the VN startup script on app update.
